# auth-client

A lightweight **TypeScript authentication client** designed to connect to a backend API and handle common authentication tasks such as:

- User registration
- User login
- Token management
- Authenticated API requests

The library acts as a simple interface between your frontend application and your authentication backend.

---

## Features

- Written in **TypeScript**
- Simple **sign-up and sign-in methods**
- Automatic **token management**
- Easy integration with frontend applications
- Minimal and framework-agnostic
- Works with any backend that exposes a basic authentication API
